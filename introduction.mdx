## What are Antigravity Awesome Skills?

Antigravity Awesome Skills is a curated library of **968+ high-performance agentic skills** designed to work seamlessly across all major AI coding assistants. Skills are specialized instruction files (markdown) that teach AI agents how to perform specific tasks perfectly, every time.

**Think of it like this:** Your AI is a brilliant intern. Skills are the SOPs (Standard Operating Procedures) that transform them into a Senior Engineer.

## How skills work

Once installed, skills are invoked naturally in conversation with your AI assistant:

<CodeGroup>

```bash Claude Code
>> Use @brainstorming to help me design a todo app
```

```bash Cursor
@brainstorming help me design a todo app
```

```bash Gemini CLI
Use the brainstorming skill to help me plan my app
```

```bash Antigravity IDE
Use @brainstorming to plan this feature
```

</CodeGroup>

<Tip>
  Most modern tools use the `@skill-name` syntax. When in doubt, try that first!
</Tip>

## Trust and safety

Skills are classified by risk level so you know what you're running:

- 🟣 **Official**: Maintained by Anthropic/Google/Microsoft/Vendors (High Trust)
- 🔵 **Safe**: Community skills that are non-destructive (Read-only/Planning)
- 🔴 **Risk**: Skills that modify systems or perform security tests (Authorized Use Only)
