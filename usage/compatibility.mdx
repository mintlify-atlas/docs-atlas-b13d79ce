## Universal SKILL.md Format

These skills follow the universal **SKILL.md** format and work with any AI coding assistant that supports agentic skills.

<Note>
All 968+ skills in Antigravity Awesome Skills use a standardized format that makes them compatible across different AI platforms.
</Note>

## Platform Compatibility Table

| Tool | Type | Status | Invocation Example | Default Path |
|------|------|--------|-------------------|-------------|
| **Claude Code** | CLI | ✅ Full Support | `>> /skill-name help me...` | `.claude/skills/` |
| **Gemini CLI** | CLI | ✅ Full Support | `(User Prompt) Use skill-name...` | `.gemini/skills/` |
| **Codex CLI** | CLI | ✅ Full Support | `(User Prompt) Use skill-name...` | `.codex/skills/` |
| **Kiro CLI** | CLI | ✅ Full Support | `(Auto) Skills load on-demand` | Global: `~/.kiro/skills/` · Workspace: `.kiro/skills/` |
| **Kiro IDE** | IDE | ✅ Full Support | `/skill-name or (Auto)` | Global: `~/.kiro/skills/` · Workspace: `.kiro/skills/` |
| **Antigravity** | IDE | ✅ Native | `(Agent Mode) Use skill...` | Global: `~/.gemini/antigravity/skills/` · Workspace: `.agent/skills/` |
| **Cursor** | IDE | ✅ Native | `@skill-name (in Chat)` | `.cursor/skills/` |
| **GitHub Copilot** | Extension | ⚠️ Text Only | `(Paste content manually)` | N/A |
| **OpenCode** | CLI | ✅ Full Support | `opencode run @skill-name` | `.agents/skills/` |
| **AdaL CLI** | CLI | ✅ Full Support | `(Auto) Skills load on-demand` | `.adal/skills/` |

<Warning>
**Default installer path**: `~/.gemini/antigravity/skills` (Antigravity global). Use `--path ~/.agent/skills` for workspace-specific install. For manual clone, `.agent/skills/` works as workspace path for Antigravity.
</Warning>

<Tip>
**OpenCode Path Update**: OpenCode path is changed to `.agents/skills` for global skills. See the [Place Files](https://opencode.ai/docs/skills/#place-files) directive on OpenCode Docs.
</Tip>

<Tab title="Claude Code">

### Claude Code (Anthropic CLI)

**Installation:**

```bash
# Using npx
npx antigravity-awesome-skills --claude

# Using git clone
git clone https://github.com/sickn33/antigravity-awesome-skills.git .claude/skills
```

**Invocation Pattern:**

```bash
# In your terminal with Claude Code:
>> /skill-name help me design a feature

# Alternative syntax:
>> Use @skill-name to review my code
```

**Example:**

```bash
>> /brainstorming help me plan a SaaS MVP
>> Use @api-security-best-practices to audit my endpoints
```

## Installation Options

### NPX (Recommended)

The npx installer automatically detects or uses the specified path:

```bash
# Default: ~/.gemini/antigravity/skills (Antigravity global)
npx antigravity-awesome-skills

# Tool-specific installations
npx antigravity-awesome-skills --antigravity  # Same as default
npx antigravity-awesome-skills --claude       # .claude/skills/
npx antigravity-awesome-skills --gemini       # .gemini/skills/
npx antigravity-awesome-skills --codex        # .codex/skills/
npx antigravity-awesome-skills --cursor       # .cursor/skills/
npx antigravity-awesome-skills --kiro         # ~/.kiro/skills/

# Custom path
npx antigravity-awesome-skills --path ./my-skills
npx antigravity-awesome-skills --path .agents/skills  # OpenCode
npx antigravity-awesome-skills --path .adal/skills    # AdaL CLI

# Workspace-specific
npx antigravity-awesome-skills --path ~/.agent/skills
```

<Tip>
Run `npx antigravity-awesome-skills --help` for all options.
</Tip>

### Git Clone

For manual installation or custom paths:

```bash
# Antigravity global (matches npx default)
git clone https://github.com/sickn33/antigravity-awesome-skills.git ~/.gemini/antigravity/skills

# Workspace-specific
git clone https://github.com/sickn33/antigravity-awesome-skills.git .agent/skills

# Tool-specific
git clone https://github.com/sickn33/antigravity-awesome-skills.git .claude/skills
git clone https://github.com/sickn33/antigravity-awesome-skills.git .gemini/skills
git clone https://github.com/sickn33/antigravity-awesome-skills.git .cursor/skills

# Custom location
git clone https://github.com/sickn33/antigravity-awesome-skills.git ~/my-custom-path
```

## Updating Skills

If the directory already exists, the installer runs `git pull` to update:

```bash
# Using npx (automatic update)
npx antigravity-awesome-skills

# Manual update
cd ~/.gemini/antigravity/skills  # or your installation path
git pull origin main
```
